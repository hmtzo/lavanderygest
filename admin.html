    🔗 Como conectar seu Google/Microsoft

    Google Calendar: abra calendar.google.com → Configurações do calendário → Integrar calendário → copie o URL secreto no formato iCal.

    Microsoft Outlook / 365: em Outlook web → Configurações → Calendário → Calendários compartilhados → Publicar um calendário → selecione permissão "Ver todos os detalhes" → copie o link .ics.

    Cole esses URLs (separados por vírgula) na aba Integrações → card "Agenda externa (iCal)" e clique em Salvar. Os eventos aparecem aqui automaticamente.

    Repasse Inteligente

    ⚙️ Configurar valores por condomínio
    Preço máquina: quanto o usuário paga. Repasse/ciclo: quanto o condomínio recebe. Imposto: % sobre o repasse.
    📥 Importar tarifas de planilha Google Sheets
    Cole o link da sua planilha com 1 aba por condomínio — o sistema lê cada aba, extrai REEMBOLSO POR CICLO, ICMS e Preço máquina, e configura tudo automaticamente.
    📅 Lançar repasses do mês
    Escolha o mês, digite os ciclos de cada condomínio — o repasse é calculado na hora usando o valor configurado acima. Clica em "Salvar tudo" quando terminar.

    🏗 Notificações de Implantação

    Quais eventos disparam e-mail/WhatsApp pros admins cadastrados. O condomínio é notificado em eventos relevantes pra ele (status, prazo, conclusão).

    📧 E-mail · Microsoft 365 (Outlook / Exchange Online)

    Envio via SMTP TLS do Office 365. Se sua conta tem MFA, crie uma senha de aplicativo em vez da senha da conta.

    📱 WhatsApp Web (não-oficial)
    —

    Conexão via Baileys — escaneie o QR uma vez com o celular que vai receber/enviar mensagens. A sessão fica salva e reconecta sozinha.

    Desconectado
    Clique em Conectar para gerar o QR. No WhatsApp do seu celular: Configurações → Aparelhos conectados → Conectar um aparelho.
    🗺️ Google Maps

    Roteirização real por ruas + distâncias/tempo. Habilite Geocoding API e Distance Matrix API no console do Google.

    🗂️ Google Drive — fotos (grátis, 15 GB)

    As fotos ficam numa pasta do seu Google Drive. Precisa de 3 passos: 1) habilitar a Drive API no seu projeto Cloud; 2) criar uma conta de serviço em IAM → Service Accounts e gerar uma chave JSON; 3) criar uma pasta no Drive, compartilhar com o e-mail da service account (role Editor), e colar o ID da pasta (parte depois de /folders/ na URL).

    💡 Ao finalizar uma visita, as fotos são enviadas para essa pasta. O link público (qualquer pessoa com o link) é salvo junto com a visita e aparece no relatório PDF.
